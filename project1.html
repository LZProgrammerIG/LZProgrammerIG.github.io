Project One

Action Platformer | Unreal Engine | C++ | Blueprints

Overview

This is a fast-paced action platformer built in Unreal Engine 5 using C++ and Blueprints. Core mechanics include double-jump, enemy AI, and level transitions.

Key Features

  • Custom movement controller in C++
  • Enemy AI using behavior trees
  • Dynamic camera system with shake & zoom
